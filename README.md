# EchoChamber
Эзотерический язык программирования, основанный на концепции акустики и затухания звука. Здесь данные — это не просто числа, а громкость, которая исчезает с каждым мгновением | An esoteric programming language based on the concepts of acoustics and sound attenuation. Here, data isn't just numbers, but volume, fading with each passing moment
